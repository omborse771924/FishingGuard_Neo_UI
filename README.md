# FishingGuard Neo UI

FishingGuard Neo UI is a modern phishing detection web application interface designed to help users identify suspicious and potentially malicious links through an interactive, visually engaging dashboard. The project focuses on cybersecurity awareness, intuitive design, and a smooth user experience.

## Features

- Modern and responsive phishing detection UI
- Clean home page design
- User login interface
- Scan interface for phishing URL input
- Analysis screen for displaying detection results
- Dashboard screen for user insights and monitoring
- Interactive and visually appealing design
- Built for cybersecurity-themed web experiences

## Project Objective

The goal of this project is to provide a user-friendly web interface for a phishing detection system. It is designed to make phishing analysis simple, accessible, and visually understandable for users. The interface can be connected with a machine learning or backend-powered phishing detection engine for real-time threat analysis.

## Tech Stack

- React.js
- JavaScript
- HTML5
- CSS3 / Tailwind CSS
- Framer Motion
